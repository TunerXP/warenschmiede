Dein digitaler Werkzeugkasten

Programme, die deine Ideen Realität werden lassen

KI ist nur der Motor – diese Programme sind das Lenkrad und die Räder. Hier erfährst du nicht nur, was du installieren solltest, sondern wie diese Werkzeuge zusammenspielen, um Ergebnisse wie auf der Seite „So arbeitest du mit KI“ zu erzielen.

Zu den Praxis-Beispielen

1. Der Bau-Workflow (Code & Web)

So entsteht eine Webseite oder Software: Du holst dir den Baustein aus dem Browser und baust ihn im Editor zusammen.

Die Kommandozentrale (Browser)

Was das ist: Dein Fenster zum Internet (Chrome, Firefox, Edge).

Die Verbindung zur KI: Hier läuft dein Chat mit der KI (z.B. ChatGPT oder Gemini). Hier formulierst du deine Wünsche und kopierst die Antworten. Gleichzeitig nutzt du Dienste wie GitHub, um Code von anderen anzusehen.

Mein Tipp: Gewöhne dir an, zwei Fenster nebeneinander zu nutzen: Links der Chat mit der KI, rechts das Ergebnis.

Visual Studio Code (VS Code)

Was das ist: Ein Text-Editor für Programmiercode. Denk daran wie an Word, aber für Software.

Die Verbindung zur KI: Die KI liefert dir den "Rohtext" (den Code). Diesen kopierst du in VS Code. Erst hier wird aus dem Text eine echte Datei (z.B. eine .html Datei), die dein Computer ausführen kann.

Warum du es brauchst: Ohne Editor bleibt der Code nur Text im Chatfenster. VS Code hilft dir zudem mit Farben, Fehler leichter zu erkennen.

FileZilla (FTP-Client)

Was das ist: Ein digitales Postauto, das Dateien von deinem PC ins Internet bringt.

Die Verbindung zur KI: Wenn deine Webseite auf deinem PC (in VS Code) fertig ist, sehen sie nur du. Mit FileZilla lädst du die Dateien auf einen Server (z.B. bei IONOS), damit die ganze Welt dein KI-Projekt sehen kann.

2. Das Medien-Studio (Bild & Ton)

KI erstellt Inhalte, aber oft musst du sie schneiden, anpassen oder verfeinern.

Audacity

Was das ist: Ein kostenloses Programm zum Schneiden und Bearbeiten von Audiodateien.

Die Verbindung zur KI: Wenn du dir von einer Musik-KI (wie Suno) einen Song erstellen lässt, ist das Ende vielleicht abrupt oder zu lang. In Audacity lädst du die Datei, schneidest das Ende sauber ab oder blendest es aus (Fade-Out).

GIMP / Paint.NET

Was das ist: Bildbearbeitungsprogramme (die kostenlose Alternative zu Photoshop).

Die Verbindung zur KI: KI-Bilder haben oft kleine Fehler (z.B. Textsalat oder seltsame Finger). Hier kannst du diese Fehler retuschieren, Texte über das Bild legen oder das Format für deine Webseite zuschneiden.

Ordnung halten

3. Dein zweites Gehirn

Wer viel mit KI arbeitet, sammelt tausende gute Prompts (Befehle). Die darfst du nicht verlieren.

Obsidian / Notion

Was das ist: Digitale Notizbücher, in denen du Informationen strukturieren kannst.

Die Verbindung zur KI: Wenn ein Prompt (Befehl) super funktioniert hat, speichere ihn hier ab. Lege dir eine eigene Bibliothek an: "Prompts für Bilder", "Prompts für Code". So musst du das Rad nicht jedes Mal neu erfinden.
